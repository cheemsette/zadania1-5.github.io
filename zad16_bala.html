1. Opis znacznika DIV

    2. Element blokowy, używany to podziału stron na elementy blokowe, jeden z najczęściej używanych tagów, najczęściej używany z atrybutami class oraz id lub align, który pozycjonuje div w wybranej części ekranu.

  2. Opis znacznika SPAN

    3. Elemt liniowy, używany do frazowania treści, grupowania tekstu, i stylizowania go. Obejmowany przez globalne atrybuty.

  3. Różnica DIV i SPAN

    4. Pierwszą, podstawową różnicą jest fakt iż element DIV jest elementem blokowym, a SPAN - liniowym. Funkcją DIV-a jest sortowanie elementów HTML w "bloki", a funkcją SPAN-u jest sortowanie tekstu i jego stylizacja.

  4. Elementy liniowe, blokowe, i mieszane.

    5. Przykładem elementu blokowego jest wyżej pokazany i opisany DIV.
    Przykładem elementu liniowego jest wyżej pokazany i opisany SPAN.
