1. Gdzie jest wstawiany wewn. arkusz stylów?

    2. W sekcji head.

  2. Składnia teoretyczna

    3. < style>
    selektor { cecha: wartość; cecha2: wartość2... }
    selektor2 { cecha: wartość; cecha2: wartość2... }
    (...)
    < /style>

  3. Co może być selektorem,

    4. Praktycznie dowolny znacznik, np. p (akapit), h1 (nagłówek), td (komórka tabeli) i inne. To właśnie elementom znajdującym się pomiędzy tymi znacznikami, nadajemy atrybuty formatowania.

  4. Jakie mogą być cechy i wartości?

    5. Jako cechę, możemy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (np. kolor tekstu - color).
    Jako wartość, możemy wpisać dokładną wartość atrybutu (np. dla koloru tekstu będzie to: red, blue czy też #31F4A5 itd.). Wartość zależy ściśle od cechy, po której stoi w deklaracji.
