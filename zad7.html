1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego,

    2. Pozwala na nadanie formatowania konkretnemu pojedynczemu elementowi strony. O tym, który to będzie element, decyduje słowo kluczowe "selektor".

  2. Składnia teoretyczna ( z użyciem ramki otaczającej może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak aby instrukcja CSS nie wykonywała się a tylko wyświetlała wewnątrz ramki [  spacja niełamana] przykład „znaku specjalnego”),

    3. < selektor style="cecha1: wartość1; cecha2: wartość2...">...< / selektor>


    < selektor style="nasze style">treść bloku < /selektor>

  3. Co może być selektorem,

    4. Praktycznie dowolny znacznik, np. p (akapit), h1 (nagłówek), td (komórka tabeli) i inne. To właśnie elementom znajdującym się pomiędzy tymi znacznikami, nadajemy atrybuty formatowania.

  4. Co wpisujemy jako "cechę". Jakie jest znaczenie średników,

    5. Jako cechę, należy wpisać o jakie konkretnie atrybuty formatowania nam chodzi (np. kolor tekstu - color). Średniki służą do rozdzielania konretnych atrybutów formatowania.

  5. Co wpisujemy jako "wartość" (w deklaracji stylu).

    6. Jako wartość, wpisujemy dokładną wartość atrybutu (np. dla koloru tekstu będzie to: red, blue czy też #31F4A5 itd.). Wartość zależy ściśle od cechy, po której stoi w deklaracji.
